🚍 BusGo – Smart Bus Booking & Travel Management Platform

Excited to share BusGo, a full-stack bus booking and travel management platform developed using the MERN Stack (MongoDB, Express.js, React.js, Node.js).

✨ Features

🔹 User Registration & Authentication
🔹 Search and Book Bus Tickets
🔹 Real-Time Seat Availability
🔹 Interactive Seat Selection
🔹 Vendor Dashboard for Bus & Trip Management
🔹 Admin Dashboard for Platform Monitoring
🔹 Trip Scheduling with Overlap Detection
🔹 Booking Management & History
🔹 Responsive UI for All Devices
🔹 REST API Integration

🛡️ Women Safety Features

✅ Emergency SOS Button
✅ Quick Access to Emergency Contacts
✅ Location Sharing Support
✅ Dedicated Safety Assistance Options
